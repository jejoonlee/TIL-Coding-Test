# [Java] 자료구조 - 트라이



#### 트리 형태의 자료구조로 문자열을 저장, 삭제, 탐색을 빠르게 하기 위해 만들어졌다

- 비슷한 문자를 굳이 따로따로 저장할 필요가 없어진다
- 예) act 와 action을 따로 저장하는 것 보단, action을 저장하고, action 안에 있는 act를 사용할 수 있게 만드는 것이다
