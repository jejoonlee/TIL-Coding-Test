## 피보나치 수

#### 재귀로 풀이 (Runtime Error)

```python
user_input = int(input())

def fibo(num):
	if num == 0:
		return 0
	elif num == 1:
		return 1
	
	return fibo(num - 2) + fibo(num - 1)

print(fibo(user_input) % 1000000007)
```



#### 리스트로 풀이

```python
user_input = int(input())

fibo = [1, 1]

for i in range(2, user_input):
	fibo.append((fibo[-1] + fibo[-2]) % 1000000007)
	
print(fibo[user_input - 1])
```
