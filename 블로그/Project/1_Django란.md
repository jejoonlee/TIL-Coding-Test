# Django

> #### 장고는 파이썬 기반 웹 프레임워크다

- 장고는 경험이 많은 개발자들로 개발이 되었다.
- 장고는 오직 웹 개발을 효율적으로 할 수 있도록 개발된 오픈 소스다.



#### 장고는 굉장히 빠르다

- 장고는 개발자들이 개발을 최대한 빠르게 완성을 할 수 있도록 개발이 되어 있다.

#### 장고는 보안에 신경을 쓴다

- 장고는 보안에 신경을 많이 썼고, 개발자들이 흔히 보안 쪽으로 실수하는 것들을 피할 수 있도록 도와준다.

#### 장고는 확장이 가능하다

- 영향력이 있는 사이트들도 장고를 통해 유연하게 확장을 한다.
