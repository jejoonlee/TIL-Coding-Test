## SET

### 교집합, 합집합, 차집합

**교집합**

```python
set1 = set([1, 2, 3, 4, 5])
set2 = set([3, 4, 5, 6, 7])

print(set1 & set2)
# {3, 4, 5}
```

**합집합**

```python
set1 = set([1, 2, 3, 4, 5])
set2 = set([3, 4, 5, 6, 7])

print(set1 | set2)
{1, 2, 3, 4, 5, 6, 7}
```

**차집합**

```python
# set1에만 있는 숫자
set1 = set([1, 2, 3, 4, 5])
set2 = set([3, 4, 5, 6, 7])

print (set1 - set2)
# {1, 2}
```

**대칭차집합**

```python
# 교집합이 아닌 숫자들
set1 = set([1, 2, 3, 4, 5])
set2 = set([3, 4, 5, 6, 7])

print(set1 ^ set2)
# {1, 2, 6, 7}
```
