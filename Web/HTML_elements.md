## ✍️ Tag

### Main Root

| Element  | Description                                                  | Details |
| -------- | ------------------------------------------------------------ | ------- |
| `<html>` | HTML 문서의 루트(최상단 요소)를 나타낸다. 모든 다른 요소는 `<html>` 요소의 후손이다 |         |



### Metadata

> 데이터 안에 있는 데이터
>
> 즉 HTML 문서 안에 있는 데이터들을 표시

| Element   | Description                                                  | Details                                     |
| --------- | ------------------------------------------------------------ | ------------------------------------------- |
| `<base>`  | 문서 안의 모든 상대 URL이 사용할 기준 URL을 지정. 문서 당 하나의 `<base>` 요소만 존재 가능 |                                             |
| `<head>`  | 기계가 식별할 수 있는 문서 정보를 담는다.                    | `<title>`                                   |
| `<link>`  | 외부 리소스 연결. 주로 스타일 시트를 연결할 때 제일 많이 사용 된다 |                                             |
| `<meta>`  | `<base>`, `<link>`, `<script>`, `<style>`, `<title>` 같은 다른 메타 관련 요소로 나타낼 수 없는 메타 데이터를 나타낸다 | `name`, `http-equiv`, `charset`, `itemprop` |
| `<title>` | 브라우저의 제목 표시줄이나 페이지 탭에 보이는 문서 제목을 정의 |                                             |



### Section

| Element                                         | Description                                                  | Details                                                      |
| ----------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| `<body>`                                        | HTML 문서의 내용. 한 문서에는 하나의 `<body>` 요소만 존재할 수 있음 |                                                              |
| `<address>`                                     | 사람, 단체, 조직 등에 대한 연락처 정보                       | `<footer>`가 있으면 그 안에 배치할 수 있다 / 자동으로 italic으로 표현 됨 |
| `<article>`                                     | 문서, 페이지, 애플리케이션, 사이트 안에서 독립적으로 구분해 배포하거나 재사용할 수 있는 구획 | 안에 여러 개의 `<section>`이 존재할 수 있다. 사용 예제는 게시판, 블로그 글 등 |
| `<aside>`                                       | 주요 내용과 간접적으로만 연관된 부분을 나타낸다              | 주로 사이드바 혹은 콜아웃 박스로 표현                        |
| `<footer>`                                      | section content나 section root의 footer를 나타낸다           | 주로 작성자, 저작권 정보, 관련 문서 등을 내용으로 담는다     |
| `<header>`                                      | 소개 및 탐색에 도움을 주는 콘텐츠                            | 로고, 검색 폼 등                                             |
| `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`, | 6단계의 section 제목을 나타낸다                              |                                                              |
| `<main>`                                        | `<body>`의 주요 컨텐츠를 나타낸다                            |                                                              |
| `<nav>`                                         | 문서의 부분 중 현재 페이지, 내, 또는 다른 페이지로 링크를 보여주는 section |                                                              |
| `<section>`                                     | 문서의 독립적인 section. 적합한 의미를 가진 요소가 없을 때 사용 된다 |                                                              |



### Text Content

| Element        | Description                                                  | Details                                                      |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| `<blockquote>` | 안쪽의 텍스트가 긴 인용문임을 나타내는 것                    | `cite` 속성으로 인용문의 출처나 URL을 작성                   |
| `<div>`        | 플로우 컨텐츠를 위한 통용 컨테이너                           | `class`, `id` 속성으로 묶어 사용이 가능하다                  |
| `<figure>`     | 독립적인 콘텐츠. 이미지/동영상, 설명 등 하나의 단위로 참조 된다 |                                                              |
| `<figcaption>` | 부모 `<figure>` 요소가 포함하는 다른 콘텐츠에 대한 설명 혹은 범례를 나타낸다 |                                                              |
| `<li>`         | 목록의 항목을 나타낸다                                       |                                                              |
| `<ol>`         | ordered list. 정렬되고, 숫자로 목록을 표현                   | `reversed` 역순으로 배열 / `type` 항목을 셀 때 사용할 카운터 유형을 바꿀 수 있음 |
| `<ul>`         | unordered list. 정렬되지 않은 목록                           | 불릿으로 표현                                                |
| `<p>`          | paragraph. 하나의 문단을 나타낸다                            |                                                              |
| `<pre>`        | 미리 서식을 지정한 텍스트를 나타내며, HTML에 작성한 그대로 표현한다 | `<p>`는 엔터를 쳐도 태그 안에서 내용은 다 이어져 있지만, `<pre>`는 태그 내에서 새로운 줄을 생성할 수 있다 |



### Inline Text

| Element    | Description                                                  | Details                                          |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------ |
| `<a>`      | anchor / `href` 속성을 통해 다른 페이지나 같은 페이지의 어느 위치, 파일 등 다른 URL로 연결할 수 있는 하이퍼링크를 만든다 | 내용으로 링크 목적지의 설명을 나타내야 한다      |
| `<br>`     | break / 줄바꿈을 생성한다.                                   | 주소 등 줄의 구분이 중요한 내용을 작성할 때 유용 |
| `<cite>`   | 저작물의 출처를 표기. 제목을 반드시 포함해야 된다            | italic으로 표현이 된다                           |
| `<em>`     | 텍스트의 강세를 나타낸다 (*italic*). **콘텐츠 강조**         |                                                  |
| `<i>`      | 텍스트를 *italic*으로 표현. cite같은                         |                                                  |
| `<mark>`   | 하이라이트를 해준다                                          |                                                  |
| `<q>`      | 짧은 인라인 인용문을 나타낸 것. 줄 바꿈 없는 짧은 경우 적합  | 긴 인용문은 `<blockquote>`으로                   |
| `<s>`      | 글자에 취소선, 즉 글자를 가로지르는 선을 그린다              |                                                  |
| `<span>`   | 문구 내 통용 인라인 컨테이너이다. `<div>`와 유사하지만, `<div>`는 블록 레벨 요소고 `<span>`은 인라인 요소다 |                                                  |
| `<strong>` | 중요하거나 긴급한 콘텐츠를 굵은 굴씨로 표현한다 (**bold**)   |                                                  |
| `<b>`      | 관심을 끌기 위해서 사용. (**bold**) 반면 `<strong>`은 정말 중요한 부분이고, user에게 중요함을 더 표현하고 싶을 때 사용한다 |                                                  |
| `<time>`   | 시간의 특정 지점 또는 구간을 나타낸다. 적절한 검색 결과나, 알림 같은 특정 기능을 구현할 때 사용할 수 있다 | `datetime` 속성과 같이 사용                      |
| `<var>`    | 수학 표현 또는 프로그래밍에서 변수의 이름을 나타낸다. 주로 글씨체의 기울임꼴로 표시된다 |                                                  |



### Image and Multimedia

| Element   | Description                                                  | Details                      |
| --------- | ------------------------------------------------------------ | ---------------------------- |
| `<audio>` | 문서에 소리 콘텐츠를 포함할 때 사용                          | `controls` , `src` 등의 속성 |
| `<img>`   | 문서에 이미지를 넣는다                                       |                              |
| `<video>` | 비디오 플레이백을 지원하는 미디어 플레이어를 문서에 삽입     |                              |
| `<track>` | `<audio>`, `<video>`의 자식으로서, 자막 등 시간별 텍스트 트랙을 지정할 때 사용 |                              |

### Embedded Content

| Element    | Description                                                  | Details |
| ---------- | ------------------------------------------------------------ | ------- |
| `<embed>`  | 외부 어플리케이션이나 대화형 컨텐츠와의 통합점을 나타낸다    |         |
| `<iframe>` | 중첨 browsing context을 나타내는 요소로, 현재 문서 안에 다른 HTML 페이지를 삽입한다 |         |
| `<object>` | 이미지나, 중첩된 브라우저 context, 플러그인에 의해 다뤄질 수 있는 리소스와 같은 외부 리소스를 나타낸다 |         |



### Table Contents

| Element      | Description                                                  | Details                             |
| ------------ | ------------------------------------------------------------ | ----------------------------------- |
| `<caption>`  | Table의 설명 또는 제목을 나타낸다                            |                                     |
| `<col>`      | 열을 나타내고, 열에 속하는 칸에 공통된 의미를 부여한다. `<colgroup>` 안에서 찾을 수 있다 | 주로 `class`를 써서 의미를 부여한다 |
| `<colgroup>` | 열을 묶는 그룹을 정의한다                                    |                                     |
| `<table>`    | 행과 열로 이루어진 표를 나타낸다                             |                                     |
| `<tbody>`    | 표의 여러 행 `<tr>`을 묶어서 표 본문을 구성한다              |                                     |
| `<td>`       | table data / 데이터를 포함하는 표의 셀을 정의한다            |                                     |
| `<tfoot>`    | 테이블의 열을 요약하는 행들의 집합. 예) 총 금액              |                                     |
| `<th>`       | 테이블 행에 어떤 데이터가 들어가는지 데이터를 넣는 것        | defined by `scope` and `header`     |
| `<thead>`    | 열의 머리글인 행들의 집합. `<th>`가 `<thead>` 안에 들어간다  |                                     |
| `<tr>`       | 테이블의 열에 대한 데이터를 저장한다. 주로 `<td>` 가 `<tr>` 안에 들어간다 |                                     |

### Forms

| Element      | Description                                                  | Details               |
| ------------ | ------------------------------------------------------------ | --------------------- |
| `<button>`   | 클릭 가능한 버튼이 생긴다 / 외형은 CSS를 통해 바꿀 수 있다   |                       |
| `<datalist>` | 컨트롤에서 고를 수 있는 선택지를 나타내는 요소를 담는다      |                       |
| `<select>`   | 옵션 메뉴를 제공하는 컨트롤를 나타낸다                       |                       |
| `<optgroup>` | `<select>`의 옵션을 묶을 수 있다. 드롭 다운 옵션 내에, 카테고리를 넣을 수 있다 |                       |
| `<option>`   | `<datalist>`, `<select>`, `<optgroup>` 요소의 항목을 정의한다. `option`을 사용해 팝업 메뉴 등 목록에서 하나의 항목을 나타낼 수 있다 |                       |
| `<form>`     | 정보를 제출하기 위한 대화형 컨트롤을 포함하는 문서 section. 예) 검색창 같은 |                       |
| `<fieldset>` | 여러 컨트롤과 레이블을 묶을 때 사용한다                      |                       |
| `<legend>`   | `<fieldset>` 컨텐츠의 설명을 나타낸다. 질문 같은             |                       |
| `<input>`    | 웹 기반 양식에서 사용자의 데이터를 받을 수 있는 대화형 컨트롤을 생성 | [유형](#<input> 유형) |
| `<form>`     | 정보를 제출하기 위한 대화형 컨트롤을 포함하는 문서 section   |                       |
| `<output>`   | 사용자 행동의 결과를 삽입할 수 있는 컨테이너 요소            |                       |
| `<progress>` | 작업의 완료 정도를 나타낸다                                  |                       |
| `<textarea>` | 멀티라인 일반 텍스트 편집 컨트롤을 나타낸다                  |                       |





## ✍️ Attribute

| Element                    | Description                                                  | Details                                                      |
| -------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| `href` (hyper-reference)   | 문서 내 상대 URL이 사용할 기준 URL                           |                                                              |
| `rel` (relationship)       | 현재 문서와 연결한 아이템의 관계가 어떻게 되는지 설명        |                                                              |
|                            |                                                              |                                                              |
| **주로 미디어의 속성**     |                                                              |                                                              |
| `src`                      | 가져올 미디어의 URL을 작성한다                               |                                                              |
| `importance`               | 리소스의 상대적인 다운로드 중요도. auto는 설정을 안 한다는 것 | auto, high, low                                              |
| `height`                   | 미디어의 픽셀 기준 고유 높이                                 |                                                              |
| `width`                    | 미디어의 픽셀 기준 고유 너비                                 |                                                              |
| `alt`                      | alternative. 미디어가 다운로드가 안 보여질 경우, 대체로 텍스트로 설명이 된다 |                                                              |
| `autoplay`                 | 전체 오디오 또는 비디오 파일의 다운로드를 기다리지 않고 가능한 빠른 시점에 재생 |                                                              |
| `controls`                 | 지정한 경우 오디오 재생, 볼륨, 탐색, 일시 정지 컨트롤을 브라우저에서 제공 |                                                              |
| `loop`                     | 미디어가 끝날 경우, 다시 앞으로 반복해준다                   |                                                              |
| track  - `default`         | 사용자 설정이 다른 자막을 가리키지 않는다면 활성화할 기본 트랙을 나타낸다 |                                                              |
|                            |                                                              |                                                              |
| **주로 테이블 속성**       |                                                              |                                                              |
| `span`                     | `<colgroup>`이 차지할 열의 수를 나타내는 양의 정수           |                                                              |
| `rowspan`                  | 열을 합칠때 사용                                             |                                                              |
| `scope`                    | 각 행의 데이터에 대한 정보                                   |                                                              |
|                            |                                                              |                                                              |
| **양식에서 사용되는 속성** |                                                              |                                                              |
| `autocapitalize`           | 영문을 입력할 때 자동으로 대문자 변환                        | sentence (문장 첫 글자) / word (각 단어 첫 글자) / characters (모든 글자) |
| `autocomplete`             | 자동 완성된 값을 기본값으로 가질 수 있는지 나타낸다          |                                                              |



#### `<input>` 유형

| 유형             | 설명                                                         |
| ---------------- | ------------------------------------------------------------ |
| `button`         | 기본 행동이 없으며, value를 레이블로 사용하는 푸쉬 버튼      |
| `checkbox`       | 단일 값을 선택하거나 선택 해제할 수 있는 체크 박스           |
| `color`          | 색을 지정할 수 있는 컨트롤. 브라우저가 지원하는 경우, 활성화 시 색상 선택기를 열어준다 |
| `date`           | 날짜를 지정할 수 있는 컨트롤                                 |
| `datetime-local` | 날짜와 시간을 지정할 수 있는 컨트롤. 시간대는 지정할 수 없음 |
| `email`          | 이메일 주소를 편집할 수 있는 필드                            |
| `file`           | 파일을 지정할 수 있는 컨트롤. `accept` 속성을 사용하면 허용하는 파일 유형을 지정할 수 있음 |
| `hidden`         | 보이지 않지만 값은 서버로 전송하는 컨트롤                    |
| `image`          | `src` 특성에 지정한 이미지로 나타나는 시각적 제출 버튼       |
| `month`          | 연과 월을 지정할 수 있는 컨트롤                              |
| `number`         | 숫자를 입력하기 위한 컨트롤                                  |
| `password`       | 값이 가려진 한줄 텍스트 필드                                 |
| `range`          | 값이 가려진 숫자를 입력하는 컨트롤                           |
| `reset`          | 양식의 내용을 디폴트값으로 초기화하는 버튼                   |
| `search`         | 검색문자열을 입력하는 한줄 텍스트 필드                       |
| `submit`         | 양식 전송하는 버튼                                           |
| `tel`            | 전화번호를 입력하는 컨트롤                                   |
| `time`           | 시간대가 없는 시간값을 입력하는 컨트롤                       |
| `url`            | URL을 입력하는 필드                                          |
